üîê Authentication

POST
/auth/register

Register a new user account

Request Body

Field Type Required Description
name string Yes Full name of user
email string Yes Valid email address
password string Yes Minimum 6 characters 
{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "secure123"
}

Success Response 201 Created

{
  "message": "User registered successfully"
}

Error Responses

400 Bad Request - Validation failed (missing fields, invalid email, short password)

401 Unauthorized - Email already exists

POST
/auth/login

Login to existing user account

Request Body

Field Type Required Description
email string Yes Registered email address
password string Yes User password 
{
  "email": "john@example.com",
  "password": "secure123"
}

Success Response 200 OK

{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

Error Responses

400 Bad Request - Validation failed (missing fields, invalid email)

401 Unauthorized - Invalid credentials

üìã Task Management

Note: All task endpoints require authentication with Authorization: Bearer <token> header

POST
/tasks Auth Required

Create a new task

Request Body

Field Type Required Description
title string Yes Task title
description string No Task description 
{
  "title": "Complete technical test",
  "description": "Finish all API endpoints"
}

Success Response 201 Created

{
  "id": "a1b2c3d4-...",
  "title": "Complete technical test",
  "status": "TODO",
  "created_at": "2026-01-11T14:30:00.000Z"
}

Error Responses

400 Bad Request - Validation failed (missing title)

401 Unauthorized - Invalid or missing token

GET
/tasks Auth Required

Get all tasks for authenticated user with pagination

Query Parameters

Parameter Type Required Default Description
page number No 1 Page number
limit number No 10 Items per page (max 100)

Success Response 200 OK

{
  "data": [
    {
      "id": "a1b2c3d4-...",
      "title": "Complete technical test",
      "status": "TODO",
      "created_at": "2026-01-11T14:30:00.000Z"
    }
  ],
  "meta": {
    "total": 1,
    "page": 1,
    "per_page": 10,
    "last_page": 1
  }
}

Error Responses

401 Unauthorized - Invalid or missing token

PATCH
/tasks/:id Auth Required

Update task status

Path Parameters

Parameter Type Required Description
id UUID Yes Task ID

Request Body

Field Type Required Description
status string Yes Must be: TODO, IN_PROGRESS, or DONE 
{
  "status": "IN_PROGRESS"
}

Success Response 200 OK

{
  "id": "a1b2c3d4-...",
  "status": "IN_PROGRESS"
}

Error Responses

400 Bad Request - Invalid status or invalid UUID

401 Unauthorized - Invalid or missing token

403 Forbidden - Task belongs to another user

404 Not Found - Task not found

DELETE
/tasks/:id Auth Required

Delete a task

Path Parameters

Parameter Type Required Description
id UUID Yes Task ID

Success Response 200 OK

{
  "message": "Task deleted"
}

Error Responses

400 Bad Request - Invalid UUID

401 Unauthorized - Invalid or missing token

403 Forbidden - Task belongs to another user

404 Not Found - Task not found

Base URL

http://localhost:3000

Environment Variables

JWT_SECRET Required - Secret key for JWT signing
JWT_EXPIRES_IN 3600s - Token expiration time

cURL Examples

# Register
curl -X POST http://localhost:3000/auth/register \
  -H "Content-Type: application/json" \
  -d '{"name":"John Doe","email":"john@example.com","password":"secure123"}'

# Login
curl -X POST http://localhost:3000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"john@example.com","password":"secure123"}'

# Create Task (use token from login)
curl -X POST http://localhost:3000/tasks \
  -H "Authorization: Bearer <your-token>" \
  -H "Content-Type: application/json" \
  -d '{"title":"New Task","description":"Task description"}'

# Get Tasks
curl -X GET "http://localhost:3000/tasks?page=1&limit=5" \
  -H "Authorization: Bearer <your-token>"
